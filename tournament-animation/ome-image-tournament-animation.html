🍎 Apple Shortcuts Setup — one-time
Works on iPhone, iPad and Mac (Monterey+). Build these two Shortcuts once — after that, paste a copied list and run.

⭐ Shortcut 1 — "Tournament Favourites"

  1. 1Open the Shortcuts app → tap + New Shortcut
  2. 2Add action: Ask For Input — type Text, prompt "Paste filename list"
  3. 3Add action: Split Text — split by New Lines
  4. 4Add action: Repeat With Each — drag steps 5–7 inside
  5. 5Inside repeat: Find Photos → Filter: Name is Repeat Item → Limit 1
  6. 6Add action: Add to Album — album name: Tournament Favourites
  7. 7Add action: Change Photo → set Favourite to on
  8. 8End repeat. Name the Shortcut Tournament Favourites and save.

🗑 Shortcut 2 — "Tournament Delete Review"

  1. 1Create another New Shortcut
  2. 2Add action: Ask For Input — type Text, prompt "Paste delete list"
  3. 3Add action: Split Text — split by New Lines
  4. 4Add action: Repeat With Each — drag steps 5–6 inside
  5. 5Inside repeat: Find Photos → Filter: Name is Repeat Item → Limit 1
  6. 6Add action: Add to Album — album name: Review Delete
  7. 7End repeat. Name the Shortcut Tournament Delete Review and save.

🔁 Each time you use it

  1. 1Tap 📤 Copy Favourites or 📤 Copy Delete List above
  2. 2Open Shortcuts → run the matching Shortcut
  3. 3Paste the copied list when prompted → tap Done
  4. 4Shortcut loops through each filename and acts on your Photos library
